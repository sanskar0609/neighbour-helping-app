## 📦 Docker Image  
You can pull the Docker image using:  
```bash
docker pull sanskar0609/selenium-unityhub
```
 Features    
✅ Connect with neighbors for help & support.  
✅ Emergency services integration (hospitals, police, etc.).  
✅ Task posting & volunteering feature.
